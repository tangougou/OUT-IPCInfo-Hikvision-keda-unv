# Hikvision IPC Batch Snapshot Tool

A professional Python tool for batch out camera onfo from IP cameras

本项目用于海康威视 科达 宇视 IPC 海康NVR  设备批量信息导出，通过构造不同厂家的API去获取保存，适用于监控巡检与型号统计。
准备一份iplist.txt  放入py同文件夹内，txt内每行一个ip地址

---

## 🚀 Features

- Batch out from multiple IPC cameras information
- Timeout control
- Automatic out xls
- Simple configuration

---

## 🏗 Supported Devices

- Hikvision IPC
- Hikvision NVR
- UNV IPC
- Keda IPC
